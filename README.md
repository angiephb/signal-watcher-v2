# 🛡️ Signal Watcher - AI Security Dashboard

Este proyecto es un microproducto Fullstack diseñado para el monitoreo y análisis de señales de seguridad utilizando Inteligencia Artificial.

## 🏗️ Estructura del Monorepo

El proyecto está organizado de forma modular para facilitar su escalabilidad y mantenimiento:

* **/frontend**: Interfaz de usuario dinámica desarrollada con **Next.js 14**, **Tailwind CSS** y **Framer Motion**. Incluye visualización de severidad en tiempo real y una UI orientada a ciberseguridad.
* **/backend**: API robusta construida con **Node.js** y **Express**. Gestiona la lógica de negocio, la conexión con **Supabase** mediante **Prisma ORM** y la integración con **Google Gemini AI** para el análisis de logs.
